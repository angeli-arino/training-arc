# Training Arc

Training Arc is Abby's personal fitness companion for deciding whether today is a Slay Day, Modify the Quest, Recovery Episode, or Peace Mode.

## Local development

```powershell
npm.cmd run dev
```

Then open `http://127.0.0.1:5173/`.

## Checks

```powershell
npm.cmd run lint
npm.cmd test
npm.cmd run build
```
