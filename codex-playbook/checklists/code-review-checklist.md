# Code Review Checklist

## Structure

- [ ] Files are in sensible locations
- [ ] Components are reusable where appropriate
- [ ] Logic is not buried in JSX
- [ ] No unnecessary dependency was added

## Quality

- [ ] TypeScript types are clear
- [ ] No obvious duplicated logic
- [ ] No dead code
- [ ] No console logs left behind unless intentionally useful

## Styling

- [ ] Uses existing styling conventions
- [ ] Does not scatter random colours
- [ ] Supports light and dark mode
- [ ] Does not break existing screens

## Checks

- [ ] Lint passed, or issue explained
- [ ] Typecheck passed, or issue explained
- [ ] Tests passed, or issue explained
- [ ] Build passed, or issue explained
