# Commit Guidelines

Use small commits.

## Commit message examples

```text
feat: add body scan circular score buttons
feat: build Training Compass card
style: refine blush theme tokens
fix: remove slider from body scan
refactor: extract body scan decision logic
```

## Good PR/change summary

Include:

- what changed
- why it changed
- screenshots if UI changed
- checks run
- design checklist result
