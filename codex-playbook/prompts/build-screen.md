# Prompt — Build or Redesign a Screen

```text
Read AI_RULES.md first.

Build/redesign this Training Arc screen:
[screen name]

Use the source of truth files:
- /design/design-system.md
- /design/ui-guidelines.md
- /design/components.md
- /design/screens.md
- /design/copywriting.md
- /codex-playbook/ui-implementation-rules.md
- /codex-playbook/component-contracts.md

Requirements:
- mobile-first
- cute, soft, warm, polished
- no plum/purple dominance
- no sliders
- no admin dashboard layout
- no chunky gamer UI
- reuse existing components where possible
- create reusable components only when needed

After implementation:
- run available checks
- review against /codex-playbook/checklists/design-review-checklist.md
- fix obvious design violations
- summarize changed files and checks run
```
