# Prompt — Fix UI Drift

Use this when Codex makes something ugly, boxy, purple, corporate, or gamer-menu-ish.

```text
Read AI_RULES.md and the design review checklist.

The current UI has drifted from the Training Arc design direction.

Review the affected screen/component and fix it so that it matches:
- strawberry milk / blush / cream / peach light mode
- deep rose / cherry cocoa / warm berry dark mode
- mobile-first lifestyle app feel
- rounded soft cards
- pill buttons
- circular score buttons
- intentional small icons

Do not make it:
- plum/purple dominant
- muddy brown
- boxy
- corporate/admin-like
- chunky gamer UI
- filled with oversized decorative icons

After fixing, explain what visual drift you corrected and which files changed.
```
