# Prompt — Visual Review Before Accepting UI

```text
Read AI_RULES.md.

Review the current UI change against:
- /design/design-system.md
- /design/ui-guidelines.md
- /codex-playbook/checklists/design-review-checklist.md
- /codex-playbook/checklists/accessibility-checklist.md

Look specifically for:
- plum/purple dominance
- muddy dark mode
- boxy cards
- admin dashboard layout
- chunky gamer styling
- sliders
- oversized icons
- poor mobile spacing
- unclear primary CTA
- inconsistent components

If you find issues, fix them before final response.

In your final response, include:
- what passed
- what you fixed
- what you could not verify without a browser/screenshot
```
