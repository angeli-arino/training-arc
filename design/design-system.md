# Design System

## Design direction

Training Arc should look like a polished mobile-first lifestyle app with strawberry milk warmth and soft anime planner energy. It should feel cute, premium, and practical.

## Colour principles

- Pink, blush, cream, peach, strawberry milk, soft rose, warm ivory.
- Lavender is allowed only as a small accent.
- Avoid plum/purple as the dominant brand colour.
- Avoid muddy brown and cold purple in dark mode.

## Typography

Use readable, rounded, friendly typography. Avoid chunky gamer fonts.

Recommended pairing:

- Display: rounded friendly font for headings.
- Body: clean rounded sans-serif.

Suggested fonts:

- Display: `Nunito`, `Quicksand`, `Baloo 2`, or similar.
- Body: `Inter`, `Nunito Sans`, `Avenir`, or system rounded sans.

Rules:

- Headings can be playful but must remain readable.
- Body text should be calm and clean.
- Avoid all-caps except tiny labels.
- Avoid heavy black font weights.

## Shadows and borders

Use soft depth, not harsh outlines.

Light mode:

- subtle warm shadows
- faint rose/peach borders
- layered surfaces

Dark mode:

- gentle glow around selected/active items
- warm berry borders
- avoid harsh neon

## Icon style

Use the existing soft 3D pastel icon pack:

- pastel
- rounded
- glossy
- blush/cream/lavender accents
- cute but not noisy

Rules:

- Use icons to support meaning.
- Do not use huge floating icons everywhere.
- One hero illustration/icon per screen is enough.
- Small icons in chips/cards are okay.
- Avoid making the app look like stickers on a bad dashboard.

## Buttons

Primary button:

- pill shape
- blush-to-rose gradient or solid pink
- white/cream text
- soft shadow
- clear active/pressed state

Secondary button:

- cream or blush surface
- cocoa text
- subtle border

Tertiary button:

- text or soft chip style

## Inputs

Use soft controls:

- circular score buttons for numbers
- pill chips for choices
- grouped cards for questions
- no sliders

## Bottom navigation

Bottom nav should feel like a mobile lifestyle app.

- rounded floating container
- small icons
- short labels
- active item in pink
- inactive items muted cocoa/rose
- not a chunky game HUD

## Theme toggle

Theme toggle should be subtle.

- small segmented/pill toggle
- no huge sun/moon buttons
- can use tiny icons only
- must not dominate header
